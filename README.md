


# 🍰 Alexa SweetOrder Bot

[![Telegram Bot](https://img.shields.io/badge/Telegram-Bot-blue)](https://t.me/alexasweetorderbot)
[![Python](https://img.shields.io/badge/Python-3.10+-green)](https://python.org)
[![Aiogram](https://img.shields.io/badge/aiogram-3.x-blue)](https://docs.aiogram.dev)
[![FastAPI](https://img.shields.io/badge/FastAPI-0.104+-green)](https://fastapi.tiangolo.com)

Полноценный Telegram бот для пекарни с конструктором тортов, AI-генерацией дизайнов и системой лояльности.

## ✨ Возможности

### 🤖 Для клиентов
- 📋 **Меню** - просмотр тортов, выпечки, напитков по категориям
- 🎨 **Конструктор торта** - собери свой уникальный торт
- 🤖 **AI-генерация** - создай дизайн торта по описанию (/imagine)
- 📦 **Заказы** - быстрое оформление, отслеживание статуса
- 🔔 **Напоминания** - уведомления о готовности заказа
- 🎁 **Бонусы** - накопительная система, QR-коды на скидку
- 👥 **Рефералы** - приглашай друзей и получай бонусы
- 🎂 **День рождения** - персональный подарок

### 👑 Для администратора
- 📊 **Статистика** - аналитика заказов и клиентов
- 🍰 **Управление меню** - добавляй/удаляй товары
- 📸 **Галерея** - загружай фото работ
- 📢 **Рассылки** - общайся с клиентами
- 📈 **Отчеты** - ежедневная статистика в Telegram

## 🛠 Технологии

- **Python 3.10+** - основной язык
- **Aiogram 3.x** - фреймворк для Telegram бота
- **FastAPI** - backend API
- **PostgreSQL** - база данных
- **Redis** - кэширование и сессии
- **APScheduler** - планировщик задач
- **Docker** - контейнеризация
